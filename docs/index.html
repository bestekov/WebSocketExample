WebSocket Example

This project was built as a proof of concept to use the C# ClientWebSocket to replace polling with WebRequest for the FreePBX_Utility caller id notification.

The project runs on a Raspberry Pi with the RasPBX image. This should run on Raspbian images as well as any Debian 10 (Buster) distribution.
